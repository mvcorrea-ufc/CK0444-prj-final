 ## **Instruções para Execução da Pipeline de Modelos**
1. Instale o Poetry:
```bash
pip install poetry
```
2. Instale as dependências necessárias para rodar a pipeline de modelos: 
```bash
poetry install
```
2. Execute a pipeline e dados:
```bash
poetry run python pipeline.py <Dataset File Path>
```
